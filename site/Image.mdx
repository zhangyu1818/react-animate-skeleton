# `<Image />`

`<Image />`组件用来模拟图片区域

```jsx
<SkeletonProvider>
  <Skeleton width={500}>
    <Image height={200} padding={40} />
    <Section padding={20} gap={20} lineSize={20} lines={4} />
  </Skeleton>
</SkeletonProvider>
```

## Props

| 字段名    | 描述             | 类型             | 默认值 |
| --------- | ---------------- | ---------------- | ------ |
| width     | 宽度             | string \| number | 无     |
| height    | 高度             |string \| number | 无    |
| padding    | 内边距             |string \| number | 必填    |
