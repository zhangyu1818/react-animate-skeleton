# 例子

### 列表信息

```jsx
<Skeleton>
  <Horizontal>
    <Item width={60} height={60} />
    <Section lineSize={15} lines={2} padding="11px 40px 11px 8px" gap={8} flex />
  </Horizontal>
</Skeleton>
```
<div style={{ width: "300px" }}>
<SkeletonProvider>
  <Skeleton>
    <Horizontal>
      <Item width={60} height={60} />

    </Horizontal>
  </Skeleton>
</SkeletonProvider>
</div>
