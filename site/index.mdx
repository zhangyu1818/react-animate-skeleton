# 介绍

`react-animate-skeleton`是一个使用`styled-components`的简单骨架屏

在布局简单的情况下适用，主要用在手机页面上，因为这样的动画方式，相对来说布局会比较复杂，而简单的行，列就能用一个组件实现

自定义配置包括骨架屏颜色，动画持续时间

## 安装

```shell
npm install --dev react-animate-skeleton
```

## 使用

```jsx
import { SkeletonProvider, Skeleton, Section } from "react-animate-skeleton";
```

import { SkeletonProvider, Skeleton, Section } from "../dist/index.esm.js";

```jsx
<SkeletonProvider>
  <Skeleton width={400} animationDuration={4}>
    <Section padding={0} gap={20} lines={6} lineSize={50} />
  </Skeleton>
</SkeletonProvider>
```
