# `<Horizontal />`

`<Horizontal />`本质上就是个`display: flex`的`<div />`，让其中的元素横向排列

## Props

| 字段名    | 描述             | 类型             | 默认值 |
| --------- | ---------------- | ---------------- | ------ |
| alignItems | 竖轴排列方式     | "align-items"的值      | 无     |

可以使用`stretch`让子级高度撑满
