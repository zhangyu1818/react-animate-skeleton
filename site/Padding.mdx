# `<Padding />`

`<Padding />`组件用来加边距

## Props

| 字段名     | 描述         | 类型             | 默认值 |
| ---------- | ------------ | ---------------- | ------ |
| padding    | 边间距       | string \| number | 必填   |
| flex       | 同弹性盒模型flex属性(在Horizontal组件子级时适用) | string \| boolean | 无   |
