# `<Gap />`

`<Gap />`用来增加间距

## Props

| 字段名    | 描述             | 类型             | 默认值 |
| --------- | ---------------- | ---------------- | ------ |
| className | 自定义类名       | string           | 无     |
| width     | 宽度             | string \| number | 无     |
| height    | string \| number | 必填             |        |
